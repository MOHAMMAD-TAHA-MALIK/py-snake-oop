Snake Game

A classic arcade Snake Game clone built with Python's turtle graphics module using object-oriented programming.

Features

Smooth Grid Controls: Standard keyboard arrow keys handle real-time directional movement.
Dynamic Growth System: Snake segments extend smoothly whenever the snake collides with food items.
Scoreboard System: A live score manager that automatically tracks player performance and resets upon death.
Collision Engine: Integrated boundaries and self-intersection checks trigger game resets accurately.

System Requirements & Installation

This project utilizes Python's native packages (turtle, time). No external dependencies or pip installations are required.
